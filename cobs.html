serial_protocol.cobs.decode(data: Union[str, list, bytearray]) → bytearray[source]

Decodes COBS-encoded data back into its decoded form.

This function handles input conversion before passing the data to decode_bytearray().

Parameters

data (Union[str, list, bytearray]) – The COBS-encoded data. - str: A hex string (e.g., “00 11 22”). - list: A list of integer values. - bytearray: A raw bytearray.

Returns

The decoded byte sequence.

Return type

bytearray

Raises

TypeError – If data is not one of the supported types.

serial_protocol.cobs.decode_bytearray(data: bytearray) → bytearray[source]

Decodes a COBS-encoded bytearray back to its original form.

COBS decoding reconstructs the original byte sequence by inserting zero bytes at positions indicated by the length markers.

Parameters

data (bytearray) – The COBS-encoded byte sequence.

Returns

The original decoded byte sequence.

Return type

bytearray

Raises

  • TypeError – If data is not a bytearray.

  • ValueError – If data has an invalid format: - Must have at least two elements. - Must end with a valid frame delimiter (0x00). - Cannot contain multiple frame delimiters. - Must not have invalid zero markers.

serial_protocol.cobs.encode_bytearray(data: bytearray, delimiter: int = 0) → bytearray[source]

Encodes a bytearray using Consistent Overhead Byte Stuffing (COBS).

COBS ensures that the encoded data contains no occurrences of 0x00, except for the final frame delimiter 0x00. It does this by replacing zero bytes with a length marker.

Parameters

data (bytearray) – The input bytearray to encode.

Returns

The COBS-encoded byte sequence.

Return type

bytearray

Raises

  • TypeError – If data is not a bytearray.

  • ValueError – If data is empty.
